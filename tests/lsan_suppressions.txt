# LeakSanitizer suppressions for the test suite.
#
# Ginkgo lazily builds a process-lifetime machine_topology singleton (backed by
# hwloc) the first time an Executor touches it. Those allocations live until the
# process exits and are never freed, so LeakSanitizer reports them on every
# Ginkgo-using test. They are not leaks in this library's code; suppress them so
# real leaks in our own code still surface.
leak:gko::machine_topology
leak:hwloc
leak:libhwloc
